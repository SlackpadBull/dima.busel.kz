Halyk Qorgan

«Halyk Qorgan» — мобильное приложение для защиты жителей Казахстана от телефонного мошенничества.

Главной задачей проекта было за короткий срок создать приложение, понятное людям любого возраста, в том числе пожилым пользователям. Оно помогает проверить неизвестный номер, сообщить о мошенниках и подать заявление об ущербе, не разбираясь в сложных сценариях и государственных сервисах.

Проект разработан по заказу прокуратуры Павлодарской области. Приложение использует государственные базы данных и сведения Интерпола, а также пополняет собственную базу мошеннических номеров. Если на номер поступает большое количество жалоб, он автоматически помечается как мошеннический.

Все номера вымешленны, совпадения случайны

Главная

После регистрации по номеру телефона пользователь дополнительно задаёт пароль для защиты доступа к приложению.

Главный экран построен вокруг трёх основных сценариев: проверить входящий звонок, открыть белый список проверенных номеров и посмотреть свои заявления.

Если номер уже находится в базе мошенников, приложение предупреждает об этом. Если информации о нём ещё нет, пользователь может отправить жалобу прямо из списка вызовов.

Белый список

Белый список содержит проверенные номера банков, государственных органов и других организаций.

Пользователю не нужно искать официальный номер в интернете — достаточно открыть приложение и позвонить напрямую.

Заявления

Если мошенники причинили ущерб, пользователь может подать заявление через приложение.

Все обращения собраны в одном списке. Для каждого заявления отображается текущий статус рассмотрения, поэтому пользователь всегда понимает, на каком этапе находится его обращение.

Поиск

Достаточно начать вводить номер телефона — результаты появляются сразу.

Для каждого номера отображается информация о регистрации, количество жалоб и наличие в базе мошенников. Ниже расположены наиболее востребованные номера банков и государственных организаций из белого списка.

Доступность

Интерфейс построен вокруг трёх основных сценариев: проверить номер, сообщить о мошеннике и подать заявление. Все основные функции доступны с главного экрана и не требуют сложной навигации.

Вместо подтверждающих диалогов любое действие можно отменить в течение нескольких секунд.

Такой подход позволяет отказаться от множества окон подтверждения, которые пользователи часто закрывают автоматически, не читая текст.

Для комфортной работы в тёмное время суток предусмотрена тёмная тема.
